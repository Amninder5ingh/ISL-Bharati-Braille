# ISL-Bharati-Braille
This project focuses on building an AI-based system to recognize Indian Sign Language (ISL) hand gestures and convert them into alphabet text.  ⚠️ Current Status: The project is under development. At present, it supports hand sign → alphabet recognition only. Future stages will include conversion to Bharati script and Braille.

🚀 Features (Current)
✋ Real-time hand gesture recognition
🔤 Conversion of ISL gestures into alphabet letters (A–Z)
📷 Live webcam input using OpenCV
🤖 Basic ML model for classification

🧠 Workflow
Capture hand gestures using webcam
Detect hand landmarks (MediaPipe)
Extract features (coordinates / embeddings)
Classify gesture into alphabet (A–Z)

🛠️ Tech Stack
Python
OpenCV
MediaPipe
Scikit-learn

🔄 Future Work
Convert alphabets → Bharati script
Convert Bharati → Braille
Improve accuracy using deep learning models
Add word/sentence formation

🎯 Goal
To build an inclusive system that enables communication between hearing-impaired and visually-impaired individuals.
